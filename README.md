# laporwarga-analytics_service

This is a mock report service serves as a stub for analytics service

## Running the Service

To run the service using Docker Compose:

```bash
docker compose up
```

The service will be available on port 5001.

## Endpoints

| Method | Endpoint          | Description                 |
| ------ | ----------------- | --------------------------- |
| GET    | /api/reports/list | Retrieves a list of reports |
